MarkDown语法简介
=============================

下面是Markdwown的语法概述。  

## 一、标题  

### 1 Setext形式  
代码  
```markdown
H1
===
H2
---
=和-的数量无限制，只支持两级标题。
```  

### 2 atx形式  
代码  
```markdown  
# 一级标题
## 二级标题
###### 6级标题
```  


## 二、强调  
```markdown
斜体、黑体及两者结合
*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_
```  
`>>>`  
*This text will be italic*
_This will also be italic_

**This text will be bold**
__This will also be bold__

_You **can** combine them_

## 三、列表  
### 无序列表  
```
* Item 1
* Item 2
  * Item 2a
  * Item 2b
```  
`>>>`  
* Item 1
* Item 2
  * Item 2a
  * Item 2b

### 有序列表  
```
1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b
```
`>>>`  
1. Item 1
1. Item 2
1. Item 3
   1. Item 3a
   1. Item 3b
